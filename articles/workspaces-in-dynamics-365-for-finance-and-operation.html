Workspaces in Dynamics 365 for Finance and Operations

One of the Important feature of the dynamics 365 is the personalize user workspace with different daily used reports, frequently used data and various transaction details to provide user friendly view to improve the efficiency in day to day tasks.

Workspace is design/customized as per user requirement and only add contents often required by the user. This makes dynamics 365 very popular among end user.

There are various out of box workspaces available already for different roles of the organization to be use as per our requirement and also we can modify them if required.

There is also a flexibility to create our own workspace if existing one doesn’t fit the need of the organization. Microsoft dynamics AX company has provided a very easy and convenient way to create a workspace on your own which really gives great power to the user.

Steps to create workspace:

  1. Login to Dynamics 365
  2. From the home screen → Right Click → ‘Personalize’
  3. Click on ‘Add a workspace’
  4. Now right click on the new workspace created → then click on personalize → Rename the workspace
  5. Now it is a time to add required data in the workspace so that it will be use easily without any need to navigate in the system which is really a time consuming activity.

    6. Tile

  6. Here as we are creating a workspace for purchasing manager, we will first add Tile of vendors to the workspace.
  7. Go to Procurement and sourcing → Vendors → All vendors
  8. Go to header menu ‘Options→ in the personalize section, Click on ‘Add to workspace’
  9. Select the workspace created and selects the presentation style. We have three styles available and those are List, Link and Tile. In this case we will use Tile to display vendors.
  10. Select name of the workspace tile and choose whether to display record count on the tile or not.
  11. Go back to the workspace → you will see vendor tile along with record count

    12. List

  12. Next is to add data in the form of List in then workspace.
  13. Follow the same steps as mentioned above except select the presentation style as ‘List’
  14. In this case we will add list of purchase order in the workspace.
  15. Click on configure to add list to the workspace.
  16. Select the columns you wish to display in the workspace data list
  17. Click ok to add list to the workspace.
  18. Go to the workspace to view the List.

    19. Link

  19. Now next is to add the data in the form of link in the workspace.
  20. Follow the same steps as mentioned above except select the presentation style as ‘Link’
  21. In this case we will add procurement catalog link to the workspace.
  22. Click on Configure
  23. Add name of the link for the workspace
  24. Click ok and the link will be added to workspace./li>
  25. Go to the workspace to check the Link added.
  26. In this way we can add data to the workspace in the form of Tile, List and Link.

Share the workspace

It is very simple to share your personalize workspace with other colleague or to be use in other environment without any further hassle of creating the workspace.

Simple steps to share the workspace

  1. Go to the workspace → Go to the workspace → go to header menu ‘Options’ → click on ‘Personalize this form’
  2. In the personalize menu, select Manage → click on Export
  3. System will ask for the path to save the exported file or go to download folder to locate the file. This file will be in .XML format with the name ‘User_MyWorkspaceXXX.xml’
  4. We can share this file which then can be imported to be used in some other environment or can be use by other colleagues
  5. To import the workspace, right click on the workspace area → click on ‘personalize this form’
  6. Browse the file which was shared to be imported in the system.
  7. Close and reopen Dynamics 365 to view the imported workspace.
  8. In this way we can share the workspace easily.

    9. BI Reports

    We can also add business intelligence reports in the workspace in order to provide real time operational insights.

    Follow the below steps to add BI reports to your personalize workspace:

  9. Go to the workspace → go to header menu ‘Options’ → click on ‘Personalize this form’
  10. Select the “+” symbol to add a personalization to your workspace
  11. on the right, move the selection to “yes” on the “Allow Power BI control” field
  12. Click the “Get Started” button. You will then be prompted for credentials to connect to your power BI account
  13. Select the reports you wish to add in the workspace → Click ok
  14. Reports will be added in your workspace

Final Thought

Workspace is a very powerful tool which can be used to design your space in dynamics 365 to get all the data at one place and in the style you want.
